1. JavaScript בסיסי - טיפוסי נתונים

מבוא לטיפוסי נתונים

טיפוסי נתונים או באנגלית Data Types מאפיינים ל"משפחות" את הערכים עליהם אנו מכריזים כשאנו כותבים קוד. כל ערך או נתון שייך לסוג כל שהוא או במונח המקצועי טיפוס נתון כל שהוא.

הטיפוסים הבסיסיים ב-JavaScript:

דוגמה טיפוס נתון הסבר
undefined Undefined משתנה שהוכרז אבל לא קיבל ערך
null Null ערך ריק במכוון
true / false Boolean ערכי אמת/שקר
"Hello world" String מחרוזת תווים
16 Number מספר שלם
54.3 Number מספר עשרונית
{ id: 85 } Object אובייקט עם תכונות
function fn() {} Function פונקציה
[42, "foo"] Array מערך של ערכים
/$+[a-z]^/ RegExp ביטוי רגולרי
NaN NaN "Not a Number" - לא מספר

בדיקת טיפוס נתון עם typeof

בשפת JavaScript ניתן לקבל את טיפוס הנתון של משתנה כל שהוא באמצעות הפקודה typeof. להלן מספר דוגמאות:

var price = 56.5;
// output: number
console.log(typeof price); 

var userName = "Yaron";
// output: string
console.log(typeof userName); 

var x;
// output: undefined
console.log(typeof x); 

var isSet = false;
// output: boolean
console.log(typeof isSet);

var data = [1, 2, 3];
// output: object (מערכים הם אובייקטים ב-JS)
console.log(typeof data);

var func = function() {};
// output: function
console.log(typeof func);
💡 טיפ חשוב:

מערכים ב-JavaScript מוחזרים כ-"object" על ידי typeof. כדי לבדוק אם משהו הוא מערך, השתמש ב-Array.isArray().

2. המרת נתונים

מה זה המרת נתונים?

המרת נתונים או באנגלית Type conversion או Type casting היא פעולה בה אנו ממירים משתנה או ערך מטיפוס נתון אחד לטיפוס נתון אחר כדי שנוכל לבצע עם הערך פעולות הקשורות לטיפוס נתון אחר.

דוגמה: אם נתון מחרוזת תוים עם מספר ונרצה לבצע על המחרוזת פעולות חשבון נוכל להמיר את מחרוזת התוים למספר שלם או מספר צף כדי לבצע פעולות חשבון אלו.

המרות נפוצות

1. המרה למספר שלם - parseInt()

// המרת מחרוזת למספר שלם
var x = '6';
var result = 4 + parseInt(x);
// output: 10 (ולא "46")
console.log(result);

// דוגמאות נוספות:
parseInt("123");     // 123
parseInt("123.45");  // 123 (חותך את החלק העשרוני)
parseInt("123abc");  // 123 (לוקח רק החלק המספרי)
parseInt("abc123");  // NaN (לא יכול להמיר)

2. המרה למספר עשרוני - parseFloat()

// המרת מחרוזת למספר עשרוני
var y = '87.33';
var f = parseFloat(y);
var total = f + 2;
// output: 89.33
console.log(total);

// דוגמאות נוספות:
parseFloat("123.45");    // 123.45
parseFloat("123.45abc"); // 123.45
parseFloat("abc123.45"); // NaN

3. המרה למחרוזת - String()

// המרת מספר למחרוזת
var num = 45;
var age = String(num);
// output: "string"
console.log(typeof age);

// דרכים נוספות להמרה למחרוזת:
var num2 = 123;
String(num2);    // "123"
num2.toString(); // "123"
num2 + "";       // "123" (שיטה קצרה)

המרה לערך בוליאני (Boolean)

ניתן להמיר ערכים מסוג של מחרוזת תוים או מספר או אפילו מערך וכו' לערך בוליאני של true או false באמצעות הפונקציה Boolean().

כללי Truthy ו-Falsy

ערכים שמוערכים כ-False (Falsy)
  • 0 - המספר אפס
  • "" - מחרוזת ריקה
  • null
  • undefined
  • false
  • NaN
ערכים שמוערכים כ-True (Truthy)
  • כל המספרים חוץ מ-0
  • כל המחרוזות חוץ מריקה
  • כל האובייקטים
  • כל המערכים (גם ריקים!)
  • כל הפונקציות

דוגמאות להמרה בוליאנית:

// ערכים שיהפכו ל-true
console.log(Boolean(6));        // true
console.log(Boolean(-89));      // true
console.log(Boolean('Hello'));  // true
console.log(Boolean([]));       // true (מערך ריק!)
console.log(Boolean({}));       // true (אובייקט ריק!)

// ערכים שיהפכו ל-false
console.log(Boolean(''));       // false
console.log(Boolean(0));        // false
console.log(Boolean(null));     // false
console.log(Boolean(undefined)); // false
⚠️ שימו לב:

מערך ריק [] ואובייקט ריק {} נחשבים Truthy! זה מקור שגיאות נפוץ.

3. מערכים - CRUD ואיטרציות

מבוא למערכים

מערכים הם אחד מאבני היסוד החשובים בכל שפת תיכנות ובכללם בשפת JavaScript. מערכים הם אחד ממבני הנתונים (Data Structure) הפופולריים והמקובלים בכל שפות התיכנות כמעט כולם.

הם נותנים את היכולת לאחסן מספר רב של נתונים ומידע במשתנה אחד או במקום אחד שמכיל את כל הנתונים.

פעולות CRUD

כאחד ממבני הנתונים הפופולריים על מערך לאפשר מספר פעולות יסודיות ונפוצות כשעובדים עם סוג של מבנה כל שהוא שאוגר מספר נתונים במקביל. נהוג לציין את הפעולות הללו בארבע אותיות - CRUD שהם בעצם ראשי תיבות לארבעת הפעולות הנפוצות:

  • Create - יצירת מערך חדש
  • Read - גישה לנתונים במערך
  • Update - עדכון נתונים במערך
  • Delete - מחיקת נתונים מהמערך

מבנה המערך

מערך בנוי מתאים כאשר כל תא מאחסן בתוכו ערך מסוים. הערך יכול להיות מחרוזת תוים או מספר או אפילו ערך בוליאני של true או false ולעיתים תא יכול בעצמו להכיל מערך פנימי או אובייקט.

כדי שנוכל לפנות לכל תא ותא בנפרד - כל תא מקבל "מפתח" או באנגלית key או index המאפשר פנייה ישירה לתא ספציפי מתוך כל המערך.

דוגמה ויזואלית למערך:

Index 0 1 2 3 4
Value 2 5 7 1 9

בעולם המחשבים הספירה מתחילה מ-0!

יצירת מערכים (Create)

דרך 1: Array Literal (מומלצת)

הדרך הנוחה והקלה ביותר ליצירת מערך היא ע"י שימוש באופרטור למערכים הנראה כשני סוגריים מרובעות [ ]: 

// מערך של מספרים
var nums = [10, 18, 3, 78];

// מערך מעורב (מומלץ להימנע)
var data = ['Avi', 16, "avi@gmail.com", true, null, 96.3];

// מערך ריק
var emptyArray = [];
💡 מוסכמה:

מקובל בקרב המתכנתים לקרוא למשתנה המכיל מערך בשם המרמז או מעלה קשר לערכים הנמצאים בתוך המערך (users, numbers, names וכו').

דרך 2: Constructor

// יצירת מערך עם המחלקה Array
var cars = new Array("Saab", "Volvo", "BMW");

// יצירת מערך ריק בגודל מסוים
var emptyArray = new Array(5); // מערך של 5 תאים ריקים

גישה לנתונים (Read)

var data = ["Avi", 16, true];

// גישה לערכים לפי index
console.log(data[0]); // "Avi"
console.log(data[1]); // 16
console.log(data[2]); // true

// קבלת אורך המערך
console.log(data.length); // 3

// גישה לאלמנט האחרון
console.log(data[data.length - 1]); // true

// גישה דינמית
var index = 1;
console.log(data[index]); // 16

עדכון נתונים (Update)

שינוי ערך קיים:

var users = ["Dana", "Avi", "Haim"];
users[1] = "Shimi";
console.log(users[1]); // "Shimi"
console.log(users); // ["Dana", "Shimi", "Haim"]

הוספת אלמנטים:

הוספה לסוף המערך - push()
var users = ["Dana", "Avi", "Haim"];
users.push("Shimi");
users.push("Tali");
console.log(users.length); // 5
console.log(users[4]); // "Tali"

// הוספת מספר אלמנטים בבת אחת
users.push("Ron", "Maya");
console.log(users); // ["Dana", "Avi", "Haim", "Shimi", "Tali", "Ron", "Maya"]
הוספה לתחילת המערך - unshift()
var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.unshift("Lemon", "Pineapple");
console.log(fruits[0]); // "Lemon"
console.log(fruits[1]); // "Pineapple"
console.log(fruits[2]); // "Banana"
הוספה במיקום ספציפי:
var users = ["Dana", "Avi", "Haim"];
users[3] = "Vered"; // הוספה לאינדקס 3
console.log(users.length); // 4

מחיקת ערכים (Delete)

מחיקה עם splice()

הפונקציה splice() מאפשרת למחוק אלמנטים ממיקום מסוים. הפרמטרים: מיקום התחלה, כמות אלמנטים למחוק.

// מחיקת אלמנט אחד
var numbers = [10, 20, 30, 45];
numbers.splice(2, 1); // מחק החל מאינדקס 2, אלמנט אחד
console.log(numbers.length); // 3
console.log(numbers); // [10, 20, 45]

// מחיקת מספר אלמנטים
var data = ['bmw', 'mazda', true, 63.9];
data.splice(0, 2); // מחק החל מאינדקס 0, שני אלמנטים
console.log(data.length); // 2
console.log(data); // [true, 63.9]
⚠️ חשוב לדעת:

לאחר מחיקת אלמנט מהמערך JavaScript מסדרת את המערך מחדש כאשר האלמנט שאחרי האלמנט שנמחק מקבל את האינדקס של זה שנמחק וכו'.

שיטות מחיקה נוספות:

var arr = [1, 2, 3, 4, 5];

// מחיקת האלמנט האחרון
arr.pop(); // מחזיר 5
console.log(arr); // [1, 2, 3, 4]

// מחיקת האלמנט הראשון
arr.shift(); // מחזיר 1
console.log(arr); // [2, 3, 4]

איטרציות במערכים

לא תמיד אנחנו יודעים את גודל המערך ולא את מספר האלמנטים במערך אך נרצה בכל זאת להוציא את כל הנתונים מהמערך. כדי לעשות כן יש צורך בפעולה המכונה "איטרציה" שפרושה הוא חזרה לאותו מקור מידע כדי להוציא ממקור המידע עוד ועוד נתונים.

איטרציה עם לולאת for:

var names = ['Tom', 'Bob', 'Rob', 'Jon'];
var namesLen = names.length;

for(var x = 0; x < namesLen; x++){
    console.log(names[x]);
}
/* Output in console:
Tom
Bob
Rob
Jon */

איטרציה עם לולאת while:

var names = ['Tom', 'Bob', 'Rob', 'Jon'];
var namesLen = names.length;
var i = 0;

while(i < namesLen){
    console.log(names[i]);
    i++;
}

הצגת נתונים בדפדפן:

var users = ["Avi", "Shimi", "Haim", "Moshe"];
var uLen = users.length;
var $el = document.getElementById("users-list");

for(var x = 0; x < uLen; x++){
    var item = "<li>" + users[x] + "</li>";
    $el.innerHTML += item;
}

שיטות מובנות למערכים

מיפוי המערך - map()

הפונקציה map() מאפשרת לנו בקלות יחסית לעבור על כל אחד מהאלמנטים שבמערך כדי לבצע על כולם פעולה או פעולות כל שהם ללא יוצא מן הכלל.

// הסרת רווחים מתחילת וסוף השמות
var names = [' avi', ' shimi ', 'haim', 'israel '];
names = names.map(function(val){
    return val.trim();
});
console.log(names[0].length); // 3

// הכפלת מספרים בשתיים
var numbers = [1, 2, 4, 7, 8];
numbers = numbers.map(function(num){
    return num * 2;
});
console.log(numbers[2]); // 8

פילטור המערך - filter()

הפונקציה filter() עוזרת לנו להוציא ערכים רצויים מתוך מערך של נתונים שחלקם רצויים וחלקם לא.

// סינון רק המספרים
var data = ['abc', 'df', 56, 'hij', 98.3];
data = data.filter(function(value){
    if(typeof value === 'number') return value;
});
console.log(data[0]); // 56
console.log(data[1]); // 98.3

// סינון רק מספרים זוגיים
var numbers = [3, 6, 17, 88, 91, 102];
var evens = numbers.filter(function(number){
    if(number % 2 == 0) return number;
});
console.log(evens); // [6, 88, 102]

צמצום המערך - reduce()

הפונקציה reduce() מאפשרת לצמצם את ערכי המערך לערך אחד בודד בהתאם לקריטריונים מסוימים.

var numbers = [1, 2, 3, 4];
var sum = numbers.reduce(function(total, value, index, array){
    return total + value;
});
console.log(sum); // 10

// דוגמה מתקדמת - מציאת הערך הגבוה ביותר
var maxValue = numbers.reduce(function(max, current){
    return current > max ? current : max;
});
console.log(maxValue); // 4

חיפוש במערך - find() ו-findIndex()

var data = ['simply', 'dummy', 'text', 'printing'];

// מציאת האלמנט הראשון שעונה על התנאי
var longerStr = data.find(function(str) {
    return str.length > 5;
});
console.log(longerStr); // "simply"

// מציאת אינדקס האלמנט
var index = data.findIndex(function(str) {
    return str.length > 5;
});
console.log(index); // 0

חיפוש עם indexOf()

var fruits = ["Banana", "Orange", "Apple", "Mango"];
console.log(fruits.indexOf("Apple")); // 2

// בדיקה והוספה אם לא קיים
var newFruit = "Kiwi";
if(fruits.indexOf(newFruit) == -1) {
    fruits.push(newFruit);
}
console.log(fruits[4]); // "Kiwi"

מיון המערך - sort() ו-reverse()

var fruits = ["Banana", "Orange", "Apple", "Mango"];

// מיון בסדר עולה (א-ת)
fruits.sort();
console.log(fruits[0]); // "Apple"

// מיון בסדר יורד (ת-א)
fruits.reverse();
console.log(fruits[0]); // "Orange"

// מיון מספרים (נדרש פונקציית השוואה)
var numbers = [40, 100, 1, 5, 25, 10];
numbers.sort(function(a, b) {
    return a - b; // סדר עולה
});
console.log(numbers); // [1, 5, 10, 25, 40, 100]

4. פונקציות - הגדרה ושימוש

מה היא פונקציה?

פונקציות הם אחד המרכיבים החשובים בכתיבת קוד תיכנותי ובכללם גם בשפת JavaScript. פונקציות מאפשרות שימוש חוזר באותה פרוצדורה של קוד מספר פעמים ללא צורך בחזרה על הקוד ובכתיבת אותו קוד - קוד כפול.

יתרון נוסף לפונקציה היא העובדה שפונקציה מסוגלת ל"התנהג" שונה בהתאם לערכים המועברים לפונקציה באמצעות פרמטרים וארגומנטים.

הגדרת פונקציה בסיסית

// הגדרת פונקציה פשוטה
function myFunk(){
    console.log("Hello World");
}

// קריאה לפונקציה
myFunk(); // Output: "Hello World"

פונקציות עם פרמטרים

// פונקציה עם פרמטר אחד
function greetUser(name){
    console.log("שלום " + name);
}

greetUser("אבי"); // Output: "שלום אבי"

// פונקציה עם מספר פרמטרים
function addNumbers(a, b){
    var sum = a + b;
    console.log("התוצאה היא: " + sum);
}

addNumbers(5, 3); // Output: "התוצאה היא: 8"

פונקציות עם return

// פונקציה שמחזירה ערך
function multiply(x, y){
    return x * y;
}

var result = multiply(4, 6);
console.log(result); // 24

// פונקציה שמחזירה ערך מורכב
function getFullName(firstName, lastName){
    return firstName + " " + lastName;
}

var fullName = getFullName("יוסי", "כהן");
console.log(fullName); // "יוסי כהן"

פרמטרים ברירת מחדל (ES6)

// פרמטרים עם ערכי ברירת מחדל
function greet(name = "אורח", age = 0){
    console.log("שלום " + name + ", גילך " + age);
}

greet(); // "שלום אורח, גילך 0"
greet("דנה"); // "שלום דנה, גילך 0"
greet("מיכל", 25); // "שלום מיכל, גילך 25"

דרכים שונות להגדיר פונקציות

1. Function Declaration (הכרזת פונקציה)

function sayHello(){
    return "Hello!";
}
// ניתן לקרוא לפונקציה לפני ההגדרה (hoisting)

2. Function Expression (ביטוי פונקציה)

var sayHello = function(){
    return "Hello!";
};
// לא ניתן לקרוא לפונקציה לפני ההגדרה

3. Arrow Functions (פונקציות חץ - ES6)

// תחביר קצר
var sayHello = () => "Hello!";

// עם פרמטרים
var add = (a, b) => a + b;

// עם מספר שורות

אלגוריתם FizzBuzz

FizzBuzz הוא אלגוריתם פופולרי ברקיוטינג. המטרה: להדפיס מספרים מ-1 עד 100, אבל:

  • במקום מספרים שמתחלקים ב-3: הדפס "Fizz"
  • במקום מספרים שמתחלקים ב-5: הדפס "Buzz"
  • במקום מספרים שמתחלקים גם ב-3 וגם ב-5: הדפס "FizzBuzz"

פתרון FizzBuzz:

function fizzBuzz() {
    for(var i = 1; i <= 100; i++) {
        if(i % 3 === 0 && i % 5 === 0) {
            console.log("FizzBuzz");
        } else if(i % 3 === 0) {
            console.log("Fizz");
        } else if(i % 5 === 0) {
            console.log("Buzz");
        } else {
            console.log(i);
        }
    }
}

// פתרון אלטרנטיבי - קצר יותר
function fizzBuzzShort() {
    for(var i = 1; i <= 100; i++) {
        var output = "";
        if(i % 3 === 0) output += "Fizz";
        if(i % 5 === 0) output += "Buzz";
        console.log(output || i);
    }
}

דרך 1: split().reverse().join()

function reverseString1(str) {

דרך 2: לולאת for

function reverseString2(str) {
    var reversed = "";
    for(var i = str.length - 1; i >= 0; i--) {
        reversed += str[i];
    }
    return reversed;
}

console.log(reverseString2("שלום")); // "םולש"

דרך 3: רקורסיה

function reverseString3(str) {
    // תנאי עצירה
    if(str === "") {
        return "";
    }
    
    // קריאה רקורסיבית
    return reverseString3(str.substr(1)) + str.charAt(0);
}

console.log(reverseString3("שלום")); // "םולש"

דרך 4: reduce()

function reverseString4(str) {
    return str.split("").reduce(function(reversed, char) {
        return char + reversed;
    }, "");
}

console.log(reverseString4("שלום")); // "םולש"

    return max;
}

תרגיל 5: אלגוריתם מציאת פלינדרום

כתוב פונקציה שבודקת האם מחרוזת היא פלינדרום (נקראת אותו דבר מהתחלה ומהסוף).

פתרון 
function isPalindrome(str) {
    // הפיכת המחרוזת לאותיות קטנות והסרת רווחים
    var cleanStr = str.toLowerCase().replace(/\s/g, '');
    
    // השוואה עם המחרוזת ההפוכה
    var reversed = cleanStr.split('').reverse().join('');
    
    return cleanStr === reversed;
}

// פתרון אלטרנטיבי עם לולאה
function isPalindromeLoop(str) {
    var cleanStr = str.toLowerCase().replace(/\s/g, '');
    var len = cleanStr.length;
    
    for(var i = 0; i < len / 2; i++) {
        if(cleanStr[i] !== cleanStr[len - 1 - i]) {
            return false;
        }
    }
    return true;
}

console.log(isPalindrome("אבא")); // true
console.log(isPalindrome("שלום")); // false
console.log(isPalindrome("A man a plan a canal Panama")); // true

        this.result -= number;
        return this;
    },
    
    multiply: function(number) {
        this.result *= number;
        return this;
    },
    
    getResult: function() {
        return this.result;
    },
    
    reset: function() {
        this.result = 0;
        return this;

};

העתקת אובייקטים

העתקה רדודה (Shallow Copy)

var original = {
    name: "אבי",
    age: 30,
    hobbies: ["ספורט", "מוזיקה"]
};

// Object.assign
var copy1 = Object.assign({}, original);

// Spread operator (ES6)
var copy2 = {...original};

// שינוי בעותק לא משפיע על המקור
copy1.name = "יוסי";
console.log(original.name); // "אבי"

// אבל שינוי במערכים/אובייקטים פנימיים כן משפיע!
copy1.hobbies.push("קריאה");
console.log(original.hobbies); // ["ספורט", "מוזיקה", "קריאה"]

העתקה עמוקה (Deep Copy)

// דרך פשוטה (עובדת רק עם JSON)
var original = {
    name: "אבי",
    age: 30,
    hobbies: ["ספורט", "מוזיקה"],
    address: {
        city: "תל אביב",
        street: "רחוב הרצל"
    }
};

var deepCopy = JSON.parse(JSON.stringify(original));

// שינוי בעותק לא משפיע על המקור
deepCopy.address.city = "ירושלים";
deepCopy.hobbies.push("קריאה");

console.log(original.address.city); // "תל אביב"
console.log(original.hobbies); // ["ספורט", "מוזיקה"]
💡 טיפים חשובים:
  • השוואת אובייקטים: אובייקטים מושווים לפי reference, לא לפי תוכן
  • Falsy objects: אובייקטים ריקים ({}) נחשבים truthy
  • מפתחות: מפתחות באובייקט הם תמיד strings (או Symbols ב-ES6)
  • this context: הקשר של this משתנה לפי איך קוראים לפונקציה

7. HTML5 - אלמנטים סמנטיים

מבוא ל-HTML5

HTML5 הביא איתו אלמנטים סמנטיים חדשים שמספקים משמעות ומבנה טוב יותר לתוכן הדף. האלמנטים הללו לא רק משפרים את הקריאות של הקוד, אלא גם עוזרים לכלי נגישות, מנועי חיפוש וכלים אחרים להבין טוב יותר את המבנה והתוכן.

המבנה הבסיסי של HTML5

<!doctype html>
<html lang="he" dir="rtl">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>כותרת הדף</title>
</head>
<body>
    <!-- התוכן כאן -->
</body>
</html>

אלמנטים סמנטיים עיקריים

<header> - כותרת

מטרה: מכיל תוכן פתיחה או כותרת עיקרית של הדף או קטע.

<header>
    <h1>שם האתר</h1>
    <nav>
        <ul>
            <li><a href="#home">בית</a></li>
            <li><a href="#about">אודות</a></li>
            <li><a href="#contact">צור קשר</a></li>
        </ul>
    </nav>
</header>

<nav> - ניווט

מטרה: מכיל קישורי ניווט עיקריים באתר.

<nav aria-label="ניווט ראשי">
    <ul>
        <li><a href="/" aria-current="page">דף הבית</a></li>
        <li><a href="/products">מוצרים</a></li>
        <li><a href="/services">שירותים</a></li>
        <li><a href="/blog">בלוג</a></li>
    </ul>
</nav>

<main> - תוכן עיקרי

מטרה: מכיל את התוכן העיקרי והמרכזי של הדף. שימוש יחיד בדף.

<main>
    <h1>כותרת הדף הראשית</h1>
    <p>תוכן עיקרי של הדף...</p>
    
    <section>
        <h2>קטע ראשון</h2>
        <p>תוכן הקטע...</p>
    </section>
</main>

<section> - קטע

מטרה: מגדיר קטע תמטי בתוכן עם כותרת מתאימה.

<section>
    <h2>השירותים שלנו</h2>
    <p>אנו מציעים מגוון רחב של שירותים...</p>
    
    <article>
        <h3>פיתוח אתרים</h3>
        <p>פיתוח אתרים מותאמים אישית...</p>
    </article>
</section>

<article> - מאמר

מטרה: תוכן עצמאי שיכול לעמוד בפני עצמו (מאמר, פוסט בבלוג, הודעה).

<article>
    <header>
        <h2>כותרת המאמר</h2>
        <time datetime="2023-10-15">15 באוקטובר 2023</time>
        <address>מאת: <a href="mailto:author@example.com">יוסי כהן</a></address>
    </header>
    
    <p>תוכן המאמר...</p>
    
    <footer>
        <p>תגיות: <a href="#tech">טכנולוgiה</a>, <a href="#web">פיתוח אתרים</a></p>
    </footer>
</article>

<aside> - תוכן צדדי

מטרה: תוכן צדדי הקשור לתוכן העיקרי אבל לא חלק ממנו (sidebar, פרסומות).

<aside>
    <h3>מאמרים קשורים</h3>
    <ul>
        <li><a href="#article1">מאמר קשור ראשון</a></li>
        <li><a href="#article2">מאמר קשור שני</a></li>
    </ul>
    
    <h3>הירשם לעדכונים</h3>
    <form>
        <input type="email" placeholder="כתובת אימייל">
        <button type="submit">הירשם</button>
    </form>
</aside>

<footer> - כותרת תחתונה

מטרה: מכיל מידע על הדף או הקטע, כמו זכויות יוצרים, קישורים וכו'.

<footer>
    <p>&copy; 2023 שם החברה. כל הזכויות שמורות.</p>
    <nav aria-label="ניווט פוטר">
        <ul>
            <li><a href="/privacy">מדיניות פרטיות</a></li>
            <li><a href="/terms">תנאי שימוש</a></li>
            <li><a href="/contact">צור קשר</a></li>
        </ul>
    </nav>
</footer>

    <figcaption>נלסון מנדלה</figcaption>
</figure>

<p>הפוסט פורסם ב-<time datetime="2023-10-15T14:30:00">15 באוקטובר 2023, 14:30</time></p>

<address>
